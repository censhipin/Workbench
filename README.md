# 表格数据工作台 (Table Data Workbench)

基于 AI 自然语言处理的桌面端数据处理工具。上传 Excel 表格后，通过中文自然语言指令完成排序、筛选、求和、去重、多表匹配、合并、数据清洗、公式计算、透视表等操作，还能生成图表并导出。

> 项目形态：Electron 桌面应用（兼容浏览器访问）
> 所有数据均在本地处理，不上传用户文件到外部服务器

## 全部功能

### 文件管理
- 上传 .xlsx / .xls / .csv 文件
- 多 Sheet 标签页切换
- 多文件任务管理（任意 Sheet 加入任务）

### AI 智能数据处理（12 种操作）

| 操作 | 说明 | 自然语言示例 |
|---|---|---|
| 筛选 | 按条件过滤行 | "筛选部门为技术部的数据" |
| 排序 | 按指定列升序/降序 | "按金额从大到小排序" |
| 聚合统计 | 单列/多列求和、平均、计数，支持分组 | "按部门统计总金额" |
| 匹配合并 | 多表按关键字段关联（类似 VLOOKUP） | "按姓名匹配员工表和联系方式表" |
| 合并（横向） | 多个表按行号左右拼接，短表补空 | "合并两个表" |
| 去重 | 按关键列去重 | "删除重复的订单号" |
| 选择列 | 保留/删除指定列 | "只保留订单ID和金额" |
| 数据清洗 | 移除空行、修复异常值 | "清除空行和异常数据" |
| 公式计算 | 列间四则运算 | "计算利润=金额-成本" |
| 透视表 | 分组交叉统计 | "按业务员统计各状态金额" |
| 批量更新 | 批量修改数值 | "将金额列全部乘以1.1" |
| 图表可视化 | 6 种图表 + 多格式导出 | "将结果转为柱状图" |

### 数据预览与编辑
- 锁定/编辑模式切换
- 单元格双击编辑（Enter 确认，Esc 取消）
- 拖拽行列排序
- 新增/删除行列
- 粘贴数据
- 列宽拖拽调整
- 列字母标注（Excel 风格）

### 图表可视化
- 6 种图表：柱状图、折线图、面积图、饼图（支持环形）、雷达图、散点图
- X/Y 轴选择器
- Top-N 数据筛选
- 渐变色开关
- **图表换色**：28 色预设色板 + 原生取色器自由选色
- 统计摘要栏（合计/平均/最高/最低）
- 自定义悬浮提示
- **4 种导出格式**：
  - 交互式 HTML（完整可交互，浏览器打开）
  - PNG 图片（高清 2x）
  - SVG 矢量（无限放大）
  - PDF 文档（自动横竖版）

### 数据质量检测
- 重复数据检测
- 空值/空白单元格统计
- 异常值发现（非法邮箱、手机号、身份证格式等）
- 检测后逐行查看与修复
- 异常值批量标记
- 修复后重新检测
- 内置 9 种 Verifier 验证执行结果

### 多表任务
- 任意文件/Sheet 加入任务列表
- 单文件多 Sheet 间合并/匹配
- 跨文件合并/匹配
- 匹配合并（有条件的列匹配） vs 合并（无条件的横向拼接）

### 版本管理
- 每次执行生成不可变版本
- 版本树可视化（Workflow Tree）
- 版本间自由切换
- 删除版本（含后续版本级联删除）
- 原始数据一键重置

### 数据导出
- Excel (.xlsx) — 保留样式
- CSV (.csv) — 纯文本通用
- JSON (.json) — 结构化数据

### 执行面板（右侧栏）
- **7 步执行流水线**：理解需求 → 数据分析 → 自动修复 → 验证计划 → 数据执行 → 结果验证 → 智能解释
- 每一步实时状态与耗时
- 智能解释（执行说明、自动修复摘要、警告、建议）
- 验证报告（置信度、匹配率、各项检查明细）
- 性能监控（各阶段耗时可视化）
- 修复详情（自动修复 + 建议修复）
- 数据画像（行数/列数/类型分布）
- 错误结构化展示

### 其他
- 操作历史记录与恢复
- 撤销操作
- 表格样式预设（14 种主题配色一键切换）
- 结果对比视图（原始 vs 处理后）
- 调试模式（Pipeline Trace 可视化）
- API Key 内置设置面板
- 快捷键：Enter 提交，Shift+Enter 换行

## 配置 API Key

软件依赖 DeepSeek API 进行智能语义解析。不配置 Key 时自动降级到规则匹配模式（仅支持简单关键词指令）。

1. 点击右上角 **设置 ⚙**
2. 输入你的 DeepSeek API Key
3. 点击「保存」，Key 自动存储在浏览器本地

> 获取 Key：[platform.deepseek.com](https://platform.deepseek.com) → 注册 → 创建 API Key

## 常见问题

**软件打开后空白？**
重新安装最新版本，确保正确安装。

**AI 不能理解复杂指令？**
检查 API Key 是否配置，或尝试简化指令描述。

**文件存在哪里？**
所有数据存储在浏览器 IndexedDB 中，关闭页面不丢失。

**发给别人怎么用？**
对方安装后需在设置中配置自己的 DeepSeek API Key（免费注册）。

**图表导出的 HTML 怎么用？**
双击打开即可，完全交互，鼠标悬浮查看数据。
