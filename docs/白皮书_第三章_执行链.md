## 3.2 上传文件链路

```
用户选择文件
    │
    ▼
handleAddFile()                                              [page.tsx:180]
    │
    ├── input.click() 打开文件选择器
    │
    ▼
parseFile(file)                                               [lib/file-engine.ts]
    │
    ├── xlsx.read(data, { type: 'array' })
    ├── 逐 sheet 解析：
    │   ├── 第一行为 headers → ColumnDef[]
    │   ├── 后续每行为 RowData
    │   └── 采样推断列类型 (number/text/date)
    │
    └── return WorkbenchFile { id, name, icon, sheets, rowCount, colCount, isMock }
    │
    ▼
setFiles(prev => [...prev, wf])
    ├── saveFile(wf) 到 IndexedDB
    ├── setSelectedFileId(wf.id)
    └── reset versions, currentVersionId, activeTab = 'original'
```

## 3.6 反撤销/重置链路

```
撤销 (handleUndo)
    │
    └── versions.pop() → setCurrentVersionId(remaining.last?.id)
        → 如果没版本了: setActiveTab('original')

重置 (handleReset)
    │
    ├── setVersions([])
    ├── setCurrentVersionId(null)
    ├── setActiveTab('original')
    ├── setExecutionSteps([])
    └── setError(null)
```
