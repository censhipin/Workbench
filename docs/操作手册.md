# 表格数据工作台 — 操作手册

---

## 1. 安装与启动

### 1.1 桌面版安装

1. 双击 `Table Data Workbench Setup 0.1.0.exe` 安装程序
2. 选择安装目录（默认 `C:\Program Files\Table Data Workbench` 即可）
3. 等待安装完成，桌面会自动创建快捷方式
4. 双击桌面快捷方式启动软件

### 1.2 卸载

通过 Windows 控制面板 → 程序和功能 → 找到 "Table Data Workbench" → 右键卸载

### 1.3 开发者模式（网页版）

```bash
npm run dev
```
打开浏览器访问 `http://localhost:3000`

### 1.4 桌面开发模式

```bash
npm run dev:electron
```
会同时启动 Next.js 开发服务器和 Electron 桌面窗口。

```
│  ┌───────────────┐   │  ┌────────────────────┐  │          │
│  │ 📁 文件池   ⚙  │   │  │ 数据预览区域       │  │ 执行计划  │
│  │               │   │  │（原始数据表格）     │  │          │
│  │  [上传文件]    │   │  │ Sheet1|Sheet2|...  │  │ 结果摘要  │
│  │               │   │  ├────────────────────┤  │          │
│  │  □ 销售表     │   │  │ AI 输入框          │  │ 操作历史  │
│  │  □ 员工表     │   │  │ [输入指令...]  [→] │  │          │

│  └───────────────┘   │  └────────────────────┘  │          │
│                      │                          │          │
└──────────────────────────────────────────────────────────────┘
```

### 2.1 左侧栏 — 文件池

功能：管理所有上传的 Excel 文件。

| 元素 | 说明 |
|------|------|
| 📁 文件池 + ⚙ 齿轮图标 | 标题行，齿轮图标点击打开设置弹窗 |
| [上传文件] 按钮 | 蓝色按钮，支持 .xlsx / .xls / .csv |
| 文件列表 | 每行一个文件，显示文件名和图标 |
| 文件选择按钮 📁 | 点击设为"主文件"，数据会显示在中间预览区 |
| 任务勾选框 ☐ | 勾选后文件加入"任务文件"列表（多表操作需要） |
| "共 N 个文件" | 底部文件总数统计 |

### 2.2 中间栏 — 工作区

**数据预览区域（上半部分）：**
- 显示当前选中文件的数据表格
- 支持多 Sheet 切换（表头标签页）
- 右上角按钮：编辑模式切换、数据体检
- 顶部标注"原始数据"标识

**AI 输入框（中间部分）：**
- 文本输入框：输入自然语言指令
- 快捷操作按钮：求和、排序、去重等一键快捷指令
- 快速提示词：点击自动填入示例指令
- 任务文件标签：显示已加入任务的文件

**结果预览区域（下半部分）：**
- 显示执行后的处理结果
- 右上角按钮：查看变化（前后对比）、全屏查看
- 显示结果行数列数统计
- 支持空状态展示与错误信息展示

### 2.3 右侧栏 — 任务面板

功能：展示任务执行状态和结果。

**任务卡片（顶部）：**
- 显示当前任务描述
- 状态标识：进行中（蓝色）/ 完成（绿色）/ 失败（红色）

**执行计划：**
- 5 步逐步展示：解析意图 → 验证参数 → 执行数据 → 验证结果 → 生成报告
- 每步有状态标识：等待（灰色数字）/ 执行中（蓝色旋转）/ 完成（绿色勾）/ 失败（红色叉）
- 执行中步骤有蓝色高亮背景和"执行中..."动画

**结果摘要：**
- 统计卡片：总行数、匹配数、删除数等
- 详细变更列表

**操作历史：**
- 按时间倒序排列的所有操作记录
- 每条显示操作描述、时间、涉及文件
- 点击可恢复到历史结果

## 3. 基本操作流程

### 3.1 完整流程示例：求和操作

**第一步：上传文件**
1. 点击左侧栏「上传文件」蓝色按钮
2. 在文件选择框中选择 Excel 文件（如"工资表.xlsx"）
3. 文件出现在文件池列表中

**第二步：选择文件**
1. 点击文件行左侧的 📁 按钮
2. 文件的第一个 Sheet 数据显示在中间预览区域
3. 如果有多个 Sheet，可点击表头标签切换

**第三步：输入指令**
1. 在底部 AI 输入框中输入指令，例如：
   - `计算基本工资、绩效奖金、加班补贴的总和`
   - `统计销售额`
   - `按部门汇总工资`
2. 点击输入框右侧的发送按钮（或按 Enter 键）

**第四步：查看执行过程**
1. 右侧执行计划面板逐步骤展示处理进度
2. 等待 5 个步骤全部完成（约 1-2 秒）

**第五步：查看结果**
1. 结果预览区域显示处理后的数据
2. 可点击「查看变化」对比原始数据
3. 可点击「全屏」后「导出」下载结果

### 3.2 多表操作流程（匹配/合并）

1. 上传所有需要操作的 Excel 文件
2. 勾选每个文件右侧的 ☐ 复选框（至少 2 个文件）
3. 选中其中一个作为"主文件"（📁 按钮）
4. 在 AI 输入框输入指令：
   - 匹配：`按姓名匹配两个表`
   - 合并：`合并两个表`
5. 查看执行结果

### 3.3 数据体检流程

1. 在数据预览区域点击「数据体检」按钮
2. 右侧弹出体检报告面板，包含：
   - 数据统计概览
   - 重复数据列表
   - 空值数据列表
   - 异常数据列表
   - 综合质量评分
3. 可点击「一键修复」自动处理问题
4. 可点击异常行跳转定位到具体数据

## 4. 指令参考

### 4.1 求和 / 聚合

```
统计销售额
计算基本工资、绩效奖金、加班补贴的总和
计算出总和
按部门统计工资
按产品统计销售总额
每个部门的平均工资
统计各个区域的销售数量
求最大值/最小值/平均值
```

系统支持的聚合方式：SUM（求和）、AVG（平均）、COUNT（计数）、MAX（最大）、MIN（最小）

### 4.2 排序

```

按姓名升序排列
排序工资
从低到高排列
按日期降序排列
```

### 4.3 筛选

```

筛选销售额大于 10000 的数据
只保留张三的记录
筛选日期在 2024年1月 的数据
找出工资大于 5000 的人
```

### 4.4 去重

```
去重
删除重复的姓名
去一下重
去除重复行
```

### 4.5 多表匹配

```
匹配姓名关联两个表
```

### 4.6 多表合并

```
拼接数据
```

## 5. 配置 API Key

软件依赖 DeepSeek API 进行智能语义解析。如不配置 API Key，软件会自动降级到规则匹配模式，仅支持简单的关键词指令。

**配置步骤：**
1. 点击左侧栏标题旁的齿轮图标 ⚙
2. 在弹出的设置窗口中输入你的 DeepSeek API Key
3. 点击「保存」
4. Key 会自动存储在本地，下次打开软件无需重新输入

**获取 API Key：**
1. 访问 [https://platform.deepseek.com](https://platform.deepseek.com)
2. 注册账号
3. 进入 API Keys 页面
4. 点击「创建 API Key」
5. 复制生成的 Key（以 sk- 开头）

## 6. 数据导出

1. 执行完数据处理后，结果预览区域显示结果数据
2. 点击「全屏」按钮
3. 在全屏模式下点击「导出」按钮
4. 浏览器/软件会自动下载处理后的 Excel 文件
5. 文件名格式：`processed_原文件名.xlsx`

## 7. 历史记录

每次执行数据处理操作后，系统会自动保存历史记录。

**查看历史：**
- 在右侧面板底部找到"操作历史"区域
- 按时间倒序排列，显示操作描述、时间和文件信息

**恢复历史结果：**
1. 点击任一条历史记录
2. 如果当前结果区域有数据，会弹出确认框询问是否覆盖
3. 点击确认后，历史结果数据会恢复到结果预览区域

## 8. 常见问题

### 8.1 软件打开后空白？

- 确认已正确安装最新版本
- 尝试重新安装
- 检查 Windows 系统是否为 64 位

### 8.2 AI 不能理解复杂指令？

- 检查是否已在设置中正确配置 DeepSeek API Key
- 尝试使用更简单的关键词描述，参考第 4 节的指令示例
- 确认电脑网络可以访问 DeepSeek API（api.deepseek.com）

### 8.3 上传的文件存在哪里？

所有文件数据存储在浏览器/软件内置的 IndexedDB 数据库中，关闭软件不会丢失。

如需手动清理，可以：
- 桌面版：卸载重装
- 网页版（Chrome）：开发者工具 → Application → IndexedDB → 右键删除

### 8.4 发给别人怎么用？

将安装包（`Table Data Workbench Setup 0.1.0.exe`）发给对方，对方需要：
1. Windows 64 位系统
2. 双击安装
3. 在软件设置中配置自己的 DeepSeek API Key（免费注册）

### 8.5 如何更新软件版本？

目前需要下载最新安装包重新覆盖安装。后续版本可能增加自动更新功能。
