# 数据工作台 — 交接文档

## 项目概况

表格数据工作台，Next.js 16 + React 19 + TypeScript + Tailwind CSS v4。用户上传 Excel 后用自然语言指令操作数据（排序、筛选、求和、去重、匹配、合并、公式计算等）。NLU 层优先调 DeepSeek AI，不可用时降级到规则解析。

## 当前架构

### 布局（本轮重构后的 5 栏结构）

```
TopBar (56px) — logo | 文件名—Sheet名 | version | [设置] [帮助]
├── LeftPanel (260px, bg:#f8f9fa)
│   ├── Workspace (50%) — 文件卡片列表 + 删除/勾选按钮 + sheet 子列表
│   └── Workflow (50%) — 操作步骤树 + 操作历史按钮
├── MainPanel (flex:1, bg:white)
│   ├── DataTabs (36px) — 📄原始数据 / 📊执行结果 / 📉对比
│   ├── StatusBar — 当前显示信息
│   ├── DataArea (flex:1) — DataPreview / ResultPreview / CompareView
│   └── BottomBar — 任务文件标签 + 输入框 + 执行/撤销 + 快捷操作
├── RightPanel (320px, bg:#f8f9fa)
│   ├── AI 执行计划 (人类可读/开发者模式)
│   └── 版本时间线 (竖线 + 节点卡片 + 上下文菜单)
```
